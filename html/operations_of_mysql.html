  • create table table_name(表单的名字)     //创建表单 

                        create table table_name(Field  Type  Unsigned  Null  Extra  Key  Default  comment"info" ,)

    1.Field 为字段,必填,重要;

    2.Type 定义字段类型,括号内填入定义该字段长度,必填;

    3.Unsigned 定义字段是否为有符号(负数),选填;

    4.Null 定义字段是否为空,可选,如 not null;

    5.Extra(A_I) ,即auto_increment,和主键配合使用,给主键设置自动增值;

    6.Key 设置主键(primary key),在主键中书写,其他字段不能使用;

    7.Default 设置字段默认值,选填,输入的文本即为默认值;

    8.comment"备注信息",给该字段添加备注信息;
